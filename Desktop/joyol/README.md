# Restaurant Booking Web Application

## Description
This project is a web-based application that allows users to book restaurant tables online.

## Features
- User registration and login
- Browse restaurants
- Book tables in real-time
- Admin panel for managing bookings

## Technologies Used
- Python (Flask)
- HTML, CSS
- SQLite

## How to Run
1. Clone the repository
2. Install requirements
3. Run the app
